CSS positioning cheat-sheet

In this cheat-sheet we will show an example of each CSS position type with it's code and an it's effects.

The CSS position property allows us to take html elements out of their default normal flow and move them in ways that we like. There are four types of positiong that can be applied when using the CSS position property, they are: static, relative, abosolute, and fixed.

position:static

The position of static is the same as the default normal-flow. So setting position: static has no effect from the defaults.
